# Country Explorer

A responsive country search web application built with HTML, CSS, and JavaScript using the REST Countries API.

## Features

- Search countries by name
- Filter countries by region
- View detailed country information
- Save favorite countries
- Dark/light mode
- Loading, empty, and error states
- Responsive layout

## Tech Stack

- HTML
- CSS
- JavaScript
- REST Countries API

## API

This project uses:

https://restcountries.com/v3.1/name/usa
